# README

# 项目初始化

```
npm install /  yarn
```

# 项目启动

```
npm start
```

# 项目构建

```
npm run build
```

# 项目介绍
项目技术栈使用 react + redux + react-router 
项目编写使用 rekit 插件快速创建redux相关文件，基于react 的最佳实践来创建的。
前端UI框架使用的是AntDesign
